It serves.

This is the document root the harness fetches from. If you are reading this in a browser, your server answered a real HTTP request from a real client, which is the whole point of the exercise.

  • hello.txt — a few bytes of text
  • page.html — a slightly larger page
  • bytes.bin — every byte value, NULs included
  • big.bin — 256 KB, generated by tests/mkwww.sh
  • huge.bin — 8 MB, generated too, and fetched by exactly one case: the one that makes write() go short
